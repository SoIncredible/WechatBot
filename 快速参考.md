# 快速参考指南

## 🚀 快速部署（3步搞定）

```bash
# 1. 服务器安装依赖（首次部署）
scp setup-server.sh user@server-ip:/tmp/ && ssh user@server-ip "/tmp/setup-server.sh"

# 2. 编辑deploy-bot.sh，配置服务器信息

# 3. 一键部署
./deploy-bot.sh
```

---

## 📝 常用命令速查

### 本地开发

```bash
# 启动机器人
npm start

# 持续监控日志（推荐）⭐
./view-local-logs.sh watch

# 监控面板
./view-local-logs.sh dashboard

# 查看最近日志
./view-local-logs.sh recent 50

# 查看错误
./view-local-logs.sh errors
```

### 服务器管理

```bash
# SSH登录
ssh user@server-ip

# 查看状态
monitor-bot status

# 持续监控（推荐）⭐
monitor-bot watch

# 监控面板
monitor-bot dashboard

# 启动/停止/重启
monitor-bot start
monitor-bot stop
monitor-bot restart

# 查看日志
monitor-bot recent 100
monitor-bot errors
monitor-bot search "关键词"
```

### PM2 管理（如已安装）

```bash
# 启动
pm2 start bot.js --name wechaty-bot

# 查看状态
pm2 list
pm2 show wechaty-bot

# 查看日志
pm2 logs wechaty-bot

# 重启/停止
pm2 restart wechaty-bot
pm2 stop wechaty-bot

# 开机自启
pm2 save
pm2 startup
```

### systemd 管理（如已配置）

```bash
# 启动/停止/重启
sudo systemctl start wechaty-bot
sudo systemctl stop wechaty-bot
sudo systemctl restart wechaty-bot

# 查看状态
sudo systemctl status wechaty-bot

# 查看日志
sudo journalctl -u wechaty-bot -f
sudo journalctl -u wechaty-bot -n 100

# 开机自启
sudo systemctl enable wechaty-bot
```

## 📂 重要路径

| 路径 | 说明 |
|-----|------|
| `/opt/wechaty-bot/current` | 当前运行版本 |
| `/opt/wechaty-bot/backup` | 备份版本 |
| `/opt/wechaty-bot/current/logs` | 日志目录 |
| `/opt/wechaty-bot/current/.env` | 环境变量 |

## 🐛 故障排查

### 机器人无法启动

```bash
# 1. 查看错误日志
monitor-bot errors

# 2. 检查配置
cat /opt/wechaty-bot/current/.env

# 3. 检查进程
ps aux | grep bot.js

# 4. 检查Node.js
node --version
```

### API调用失败

```bash
# 检查API Key
grep KIMI_API_KEY /opt/wechaty-bot/current/.env

# 测试网络连接
curl -I https://api.moonshot.cn

# 查看详细错误
monitor-bot recent 100 | grep ERROR
```

### 内存不足

```bash
# 查看内存使用
free -h

# 查看进程资源
top
htop

# 重启机器人释放内存
monitor-bot restart
```

## 🆘 紧急处理

### 机器人崩溃

```bash
# 快速重启
monitor-bot restart

# 或使用PM2
pm2 restart wechaty-bot

# 或使用systemd
sudo systemctl restart wechaty-bot
```

### 服务器资源耗尽

```bash
# 停止机器人
monitor-bot stop

# 清理日志
cd /opt/wechaty-bot/current/logs
rm -f *.log

# 重启
monitor-bot start
```

### 回滚到上一个版本

```bash
cd /opt/wechaty-bot
monitor-bot stop
rm -rf current
mv backup current
monitor-bot start
```
